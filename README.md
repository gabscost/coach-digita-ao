# ⌨️ Coach de Digitação

> Treine digitação por toque (touch typing) com módulos progressivos, minigames e avaliação em tempo real — diretamente no navegador.

## ✨ Funcionalidades

- **Módulos progressivos** (M1 a M6)  
  Da linha guia (`ASDF JKLÇ`) até números e símbolos de programação.
- **Modo Cego**  
  Oculta o teclado virtual para forçar a memorização das posições.
- **Feedback em tempo real**  
  Destaque de letras corretas/erradas, cursor animado e tecla ativa no teclado virtual.
- **Estatísticas detalhadas**  
  Precisão, WPM, tempo, erros mais comuns e histórico salvo no `localStorage`.
- **Minigames interativos**  
  - ⚡ Sprint de 30s  
  - 🔍 Caça ao erro  
  - 🃏 Carta certa (adivinhe a tecla)  
  - 💻 Copiar código  
  - 🎤 Ditado visual (memorização)  
  - 🧩 Sequência de teclas
- **Persistência de progresso**  
  Todas as sessões de treino ficam salvas no seu navegador.
